# Welcome to AstroRails 
## A site to log your discoveries across the cosmos.

![Homepage](lib/assets/index.png)

# Getting Started
* Clone this repository
* cd into the project folder
* Run
`bundle install && rails s`
* Navigate to http://localhost:3000
* Enjoy!
